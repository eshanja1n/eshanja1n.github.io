newgram

newgram is an Instagram clone built in Python using Flask and a REST API. Users can create, delete and edit their posts, double tap to like phones, leave and reply to comments, and follow and unfollow other users. This project is also built using React components, so any user actions are completed instantaneously and saved to sqlite3 database. Also developed infitite scroll feature, and made use of a history API to better save user browser history while using newgram. Each user account is also protected with multi-factor authentication through DUO API.

Python, JavaScript, React
